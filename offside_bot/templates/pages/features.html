Rosters

Built for coaches running FC 26 teams inside your server.

  • Create and manage rosters with caps, openings, and roster updates.
  • Coach-facing dashboard flow for add/remove/view/submit.
  • Roster listings for staff visibility and recruiting.

Related: commands in the Roster category.

Recruiting

Designed for staff teams that need a consistent free-agent intake flow and a searchable player pool.

  • Player pool search with structured free-agent profiles.
  • Free-agent listing channels and workflows for availability.
  • Staff tools to post/update indices and keep listings clean.

Related: commands in the Recruitment and Staff categories.

Clubs

For multi-club leagues and orgs that want consistent club operations.

  • Club listings and club-specific dashboards.
  • Manager workflows for managing coaches, roles, and premium tiers.
  • Pro coaches report channel for public listings (Pro).

Tournaments

Staff tooling for running tournaments end-to-end.

  • Create tournaments, register teams, generate fixtures, and publish brackets.
  • Match reporting, confirmations, forfeits, disputes, and deadlines.
  • Leaderboards and basic tournament stats.

Related: commands in the Tournament category.

Dashboards + setup

Offside deploys cleanly into a server with a consistent layout.

  • Setup wizard in the web dashboard verifies roles, channels, and permissions.
  • Dashboard embeds guide staff/managers/coaches through common flows.
  • Listing channels keep rosters/recruits/clubs visible in one place.

Analytics

Visibility into what your league produces, by module and collection.

  • Per-guild analytics view in the web dashboard.
  • Collection counts and operational checks to help troubleshoot setup.
